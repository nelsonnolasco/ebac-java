Desenvolvimento de páginas com JavaServer Pages (JSP)
O (JSP) é uma linguagem de script com especificação aberta que tem como objetivo primário a geração de conteúdo dinâmico para páginas da Internet, sendo lançada em 1999 pela Sun Microsystems sendo o JSP é semelhante ao PHP, porém usa a linguagem de programação Java. 
O JSP pode ser usado para criar páginas web, assim como a tecnologia servlet. O JSP pode ser considerado uma extensão para o servlet, pois proporciona uma divisão sobre o que são páginas e o que cabe ao servidor. 
JSP simplifica o processo de desenvolvimento da web e os programadores podem incorporar elementos dinâmicos em páginas web, porém, O objetivo da linguagem JSP não é só o desenvolvimento de páginas dinâmicas para Internet. Com ela é possível desenvolver sistemas inteiros para Internet. Além disso, existem diversos benefícios em se utilizar a linguagem JSP.
Por ser uma linguagem Server-Side necessita de um servidor para o seu funcionamento. Assim, o Apache Tomcat, desenvolvido pela Apache Software Foundation, é um servlet container de código aberto, ou seja, uma aplicação que interpreta e processa servlets (java servlets) e JSP (Java Server Pages). O servidor está disponível livremente na Internet sem a necessidade de pagamento de licenciamento e está disponível no endereço http://tomcat.apache.org para diversas plataformas, entre elas o Windows, Linux, Solaris etc.


Desenvolvimento de páginas com Angular
O Angular é um framework de desenvolvimento web de código aberto desenvolvido pelo Google. É uma plataforma que facilita a criação de aplicações web. É construído em cima do TypeScript, uma linguagem de programação que é o SuperSet do JavaScript. O TypeScript adiciona recursos como tipagem estática e suporte a classes e interfaces, o que torna o desenvolvimento em larga escala mais fácil e mais seguro. 
O Angular permite o desenvolvimento multiplataforma, acelera o desenvolvimento de aplicações web e tem um conjunto de ferramentas de desenvolvimento bastante acessíveis e poderosas. 
O Angular é uma ferramenta da Google usada por muitas empresas, visto que permite construir sites responsivos oferecendo uma estrutura sólida para desenvolvimento, incluindo suporte a gerenciamento de estado, roteamento, validação de formulários e muito mais. 
A plataforma Angular provê recursos para o desenvolvimento de Single Page Applications (SPA). Ou seja, uma única página web, com o objetivo de fornecer a experiência ao usuário parecida com a de um aplicativo de desktop. Também, este framework fornece suporte a múltiplas plataformas de desenvolvimento. consegue oferecer suporte ao desenvolvimento rápido de aplicações através de uma API simples, bem estruturada e bem documentada, o que acaba trazendo bastante produtividade. Frameworks SPA atualmente são um padrão de mercado de desenvolvimento front-end.
