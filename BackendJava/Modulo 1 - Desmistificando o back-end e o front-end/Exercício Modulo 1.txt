O back-end é a parte da programação que trata do funcionamento interno de um software, de tudo o que está por trás da interface de uma aplicação, como os sistemas, o banco de dados, a segurança de dados, o envio e recebimento de informações e o armazenamento.
É responsável por armazenar dados e garantir a segurança do site como um todo, como quando você faz login numa rede social, o back-end valida os dados e faz uma conexão com o banco de dados para tentar autenticar o login. 
O back-end trabalha em conjunto com o front-end para entregar o produto para o usuário final. O front-end se concentra nos aspectos que os usuários podem ver. Por outro lado, o back-end é tudo o que faz o aplicativo funcionar.
É desenvolvido por meio de código, por linguagens de programação como Java, Python, Ruby, Golang, dentre outras.
Exemplo: Na página da EBAC temos como back-end a estrutura de software que une o usuário à máquina, à servidores para trocas de mensagens com bancos de dados ou caminhos para repositórios dos vídeos das aulas, banco de dados com os quizes no final de cada aula e/ou módulo, de onde se baixam o material de apoio às aulas etc.

O front-end, por sua vez, é a parte visual de um site, aplicativo, software ou sistema operacional com a qual os utilizadores interagem, é o “lado do cliente”. É responsável por toda a estrutura, design, conteúdo, comportamento, desempenho e capacidade de resposta de um site ou aplicação. 
O front-end é desenvolvido por meio de código, normalmente com as tecnologias base da Web: HTML, CSS e JavaScript. Envolve a criação de uma experiência visual agradável e intuitiva para os usuários, garantindo que o site ou aplicativo seja fácil de usar e compreender.
Exemplo: Na página da EBAC temos como front-end o layout da página que proporciona uma maior facilidade do usuário acessar os materiais de apoio às aulas, assistir os vídeos das aulas, o visual do editor de texto dos exercícios, o template interessante dos quizes, diversos botões que facilitam avançar, retornar, acessar tutores e suporte quando necessário em caso de dúvidas.
